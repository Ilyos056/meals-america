# Meals America — правильное питание

Static website recreated from a [Figma design](https://www.figma.com/design/F8FYlPVL9x7jd3ThcYj0zw/) — a healthy home-delivered meals service (Florida, USA).

## Pages

- `index.html` — home (hero, about, benefits, nutrition plans, delivery, who we serve)
- `about.html` — Welcome to Meals America
- `how-it-works.html` — qualifying questions, eligibility, Aging Waiver
- `menu.html` — weekly menus (Gourmet, Comfort, Deli, International, Family, Homestyle, Breakfast, Vegetarian, Salads, Gluten-Free) + heating instructions
- `contacts.html` — contacts + Get Started form

## Tech

Plain HTML + CSS + vanilla JS. No build step — just open `index.html` in a browser.

Fonts: Comfortaa, Poppins, Caveat (Google Fonts). Photos: Unsplash.

## Run locally

Open `index.html`, or serve with:

```
npx serve .
```
